# Palencia Burger Co - Carta Digital Interactiva

Aplicación web comercial diseñada bajo un enfoque **mobile-first** para la digitalización de menús en restauración. El proyecto fue desarrollado originalmente utilizando WordPress y maquetado a medida con Elementor, optimizando la experiencia de usuario (UX/UI) en dispositivos móviles para el acceso rápido mediante códigos QR en mesa.

Para su despliegue óptimo, alta velocidad de carga y persistencia eterna, el sitio ha sido exportado a código estático (HTML5/CSS3/JS) eliminando la carga del servidor de bases de datos relacionales en producción.

## 🚀 Características Clave
* **Diseño Responsive:** Optimización completa para pantallas verticales de smartphones.
* **Componentes Visuales:** Tarjetas de producto personalizadas con efectos interactivos (*hover*).
* **Arquitectura Ligera:** Conversión de CMS dinámico a sitio estático para un rendimiento web de alto nivel.

## 🛠️ Tecnologías e Infraestructura Utilizadas
* **Core:** WordPress, PHP
* **Maquetación & UI:** Elementor, HTML5, CSS3, JavaScript
* **Entorno de Desarrollo:** LocalWP, MySQL, Servidor Nginx
* **Despliegue & Hosting:** Git, GitHub, Vercel
